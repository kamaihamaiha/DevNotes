## C++ 如何工作

`` #include <iostream> `` 语句是 preProcessor 语句（预处理指令）

### 预处理

任何以 # 开头的都是预处理指令。

### main 函数返回值

如果没有返回任何值，那么它会返回 0。

### 头文件不会被编译

头文件会被 include 到 cpp 文件中

### 每个 cpp 文件被单独编译，然后被编译成 object 文件

### 链接

链接器（linker）把 object 文件链接起来，组成可执行文件，比如 exe 文件（Windows下）。
