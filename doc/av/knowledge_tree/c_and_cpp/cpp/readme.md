#### 为什么学习C++

- 编写性能好的代码，很常用
- 为一个奇怪的架构或平台编写原生代码
- 想获取对硬件的最简单进程的控制
- 游戏领域
    - C++、unity、虚幻、frostbite 等游戏引擎
- 可以控制 CPU 的每一条指令
- 全平台可用：Windows、Mac OS、Linux、Android、IOS
