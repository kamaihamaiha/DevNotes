## Gradle 环境搭建

- 配置环境
- 实现 hello world

### 配置环境

- 确保安装了 JDK
- 安装 gradle
  - 第一种: 包管理器
  - 第二种: 手动安装

### 实现 hello world

build.gradle 是Gradle默认构建的文件

创建一个空的 folder, 并创建一个 build.gradle 文件

```groovy
task hello {
    doLast {
        println 'Hello world!'
    }
}
```

在终端执行

```shell
gradle -q hello
```


#### 代码解读

- task: 任务，独立的原子操作
- doLast: 任务执行后会执行的代码
